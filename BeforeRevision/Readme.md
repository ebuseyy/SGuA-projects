## BeforeRevision Folder Description

This folder contains the original implementations used before the revision stage of the manuscript.

### C# Implementations

* **FGA.cs**
  This file corresponds to the original implementation of the Saplings Growing up Algorithm (SGuA) developed in C#.
  It represents the pre-revision version of SGuA used in early experiments.

* **Form1.cs**
  This file corresponds to the original implementation of the League Championship Algorithm (LCA) developed in C#.
  It includes the interface-based execution used during the initial experimental phase.

Note: These implementations were developed as standalone desktop applications (Windows Forms). The algorithmic logic is embedded within the UI-based structure.

### Python Notebooks

* **ABC.ipynb, BOA.ipynb, GWO.ipynb**
  These notebooks include early-stage implementations of the respective algorithms applied to the ORC problem.

* **OzgeSGuA_CEC_InitPop.ipynb**
  This notebook includes SGuA experiments on CEC benchmark functions with different initialization strategies.

### Important Note

Some algorithms referenced in the manuscript (e.g., GA and WOA) were implemented and tested during the study; however, their original source files could not be fully recovered from the archive. Their results are still reflected in the manuscript.
